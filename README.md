# GamePlaza
## Simple e-shop with ASP.NET Core and Vue.js
Project in progress...
### Backend:
- ASP.NET Core (net8.0, C# 12.0)
- EF Core (8.0.8)
- PostgreSQL (14.12)
- Fluent Validation (11.9.2)
- Bogus (35.6.1)
### Frontend:
- Vue.js 3
- TypeScript
- Tailwindcss
- Shadcn-vue components library
- Pinia store
- Vue Router
- Vite.js
